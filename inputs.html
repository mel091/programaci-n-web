HTML Input Types

Los inputs son elementos que permiten al usuario poder interactuar con la página web a través de la entrada de datos; los podemos ver en textos, contraseñas, correos electrónicos y más.

Botón

Crea un botón sin funcionalidad, debe asociarse a algún evento en javaScript para que funcione.

Check Box

Crea una casilla de verificación que el usuario puede marcar o desmarcar. Común en formularios.

Color

Permite al usuario seleccionar un color de la paleta, al seleccionarlo, se establece el color.

Date

Permite al usuario seleccionar una fecha utilizando un calendario desplegable. El valor seleccionado se devuelve en formato YYYY-MM-DD.

DateTime-Local

Permite al usuario seleccionar una fecha y hora locales en un solo campo, mostrando un calendario y un selector de tiempo.

Email

Permite al usuario ingresar una dirección de correo electrónico. Este campo valida automáticamente que el formato del correo sea correcto, como tener un símbolo "@" y un dominio válido.

File

Permite al usuario seleccionar un archivo de su dispositivo para subirlo. Asigna un identificador único al campo, que puede ser usado para referenciarlo en JavaScript o CSS.

Hidden

Este código crea un campo de entrada oculto. No es visible para el usuario, pero su valor puede ser enviado con un formulario. Se usa comúnmente para almacenar información que necesita ser enviada al servidor pero que no necesita ser vista o modificada por el usuario.

Image

Crea un botón de envío en un formulario que muestra una imagen. El botón actúa como un botón de envío (submit), pero en lugar de texto, utiliza una imagen para que el usuario haga clic.

Month

Crea un campo de entrada para seleccionar un mes y un año.

Number

Crea un campo de entrada que permite al usuario ingresar números.

Password

Crea un campo de entrada para contraseñas. Los caracteres ingresados se ocultan con puntos o asteriscos para mantener la privacidad del usuario mientras escribe su contraseña.

Radio

Se usa para permitir al usuario seleccionar una sola opción de un grupo de opciones mutuamente exclusivas.

Range

Crea un control deslizante que permite al usuario seleccionar un valor dentro de un rango. El valor del control puede ser ajustado arrastrando el control deslizante a lo largo de una barra.

Reset

Crea un botón que, al hacer clic, restablece todos los campos de un formulario a sus valores iniciales. Es útil para permitir al usuario borrar todas las entradas y empezar de nuevo.

Search

Crea un campo de entrada diseñado para buscar texto. Ofrece una interfaz de búsqueda que incluye características como un icono de lupa y una opción para borrar el texto ingresado, facilitando la búsqueda de información en un formulario o en una página web.

Submit

Crea un botón de envío que envía el formulario en el que se encuentra. Al hacer clic en el botón, se envían los datos del formulario al servidor para su procesamiento.

Tel

Crea un campo de entrada para números de teléfono. Aunque no valida el formato del número, muestra un teclado numérico en dispositivos móviles, facilitando la entrada de números telefónicos.

Text

Campo de entrada de texto donde los usuarios pueden escribir datos alfanuméricos. Es el tipo de campo más común y se usa para capturar una variedad de información textual.

Time

Crea un campo de entrada para seleccionar una hora. Muestra un selector de tiempo que permite al usuario elegir una hora específica del día, y opcionalmente también minutos y segundos, según la implementación del navegador.

URL

Valida que el texto ingresado tenga un formato de URL válido y puede mostrar un teclado optimizado para la entrada de direcciones web en dispositivos móviles.

Week

Crea un campo de entrada para seleccionar una semana y un año. Permite al usuario elegir una semana específica dentro de un año, facilitando la entrada de datos relacionados con semanas en lugar de fechas exactas.
